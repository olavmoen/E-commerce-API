## EP - Course Assignment Back-end
.env file example:
- HOST=localhost
- ADMIN_USERNAME=root
- ADMIN_PASSWORD=password
- DATABASE_NAME=ecommercedb
- DIALECT=mysql
- PORT=3000
- ACCESS_TOKEN_SECRET=qwepoimnbzxc
---
Run application:
- Back-end:
    - cd back-end
    - npm install
    - npm start
- Front-end (new terminal): 
    - cd front-end
    - npm install
    - npm start
---
NodeJS v20.17.0
